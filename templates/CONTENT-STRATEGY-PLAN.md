## 📈 SEO Optimization Guidelines

### **On-Page SEO Checklist:**
- [ ] **Title Tag:** Include primary keyword, under 60 characters
- [ ] **Meta Description:** Compelling copy with keyword, 150-155 characters
- [ ] **H1:** Single H1 with primary keyword
- [ ] **H2/H3:** Logical structure with secondary keywords
- [ ] **URL:** Clean, keyword-rich slug
- [ ] **Internal Links:** 5-8 relevant internal links
- [ ] **External Links:** 3-5 authoritative sources
- [ ] **Images:** Optimized alt text, compressed files
- [ ] **Schema:** Article/HowTo markup where appropriate

### **Content Quality Standards:**
- **Word Count:** Minimum 1,500 words for competitive keywords
- **Readability:** Flesch score 60+ (8th grade level)
- **Keyword Density:** 1-2% for primary, 0.5-1% for secondary
- **Uniqueness:** 100% original content, no duplicate sections
- **Value:** Actionable insights, practical examples
- **Updates:** Refresh annually or when tools change

### **Technical Requirements:**
- **Loading Speed:** Under 3 seconds
- **Mobile Optimization:** Responsive design
- **Core Web Vitals:** Green scores
- **Structured Data:** Proper schema implementation
- **Social Sharing:** Open Graph tags
- **Analytics:** Proper tracking setup
